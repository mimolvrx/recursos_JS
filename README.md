# 📚 Recursos JavaScript

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

Este repositório reúne arquivos desenvolvidos durante meus estudos de **JavaScript**, com o objetivo de praticar conceitos fundamentais da linguagem e servir como material de consulta para futuras aplicações.

## 📂 Arquivos

### 📄 arrays.js
Exemplos e exercícios relacionados a arrays em JavaScript.

### 📄 callbackHOF.js
Prática com **Callbacks** e **Higher-Order Functions (HOF)**, explorando funções que recebem ou retornam outras funções.

### 📄 estruturaSetMap.js
Estudo das estruturas de dados **Set** e **Map**, suas características e principais métodos.

### 📄 funcoes.js
Exemplos envolvendo declaração, expressão e utilização de funções em JavaScript.

### 📄 metodosArrays.js
Prática dos principais métodos de arrays, como:

- `forEach()`
- `map()`
- `filter()`

### 📄 metodosObjetos.js
Estudo dos principais métodos para manipulação de objetos.

### 📄 objetos.js
Exemplos de criação, acesso e manipulação de objetos em JavaScript.

### 📁 JSON
Arquivos e exemplos relacionados ao formato JSON, utilizados para armazenamento e troca de dados.
